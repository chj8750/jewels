Figma 스타일 반영 목업

Figma MCP 3845 포트는 열리지 않아 레이어 API는 못 붙었지만, 로그인된 Figma 화면을 직접 확인해 기존 게임 디자인 톤을 반영했습니다. 연한 하늘색 배경, 두꺼운 흰색/검정 테두리 로고, 핑크·블루 캐릭터, 흰색 카드와 검정 CTA 스타일로 재작업했습니다.

1. 발굴기존 가로세로 땅따먹기 톤과 맞춘 캐주얼 게임형 첫 화면
2. 보석받기광고 시청 후 보석함 저장 플로우 유지
3. 보석함1억원 진행률과 교환 CTA를 게임 스타일 카드로 정리
4. 교환 완료토스포인트 교환 후 기념 보상까지 확인
정밀한 픽셀 매칭은 MCP/Export가 필요하지만, 현재 버전은 Figma 파일의 시각 톤을 기준으로 기존 파란 그라데이션 목업을 전면 교체한 버전입니다.
